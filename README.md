## How To Use :closed_lock_with_key:

```bash
# Clone this repository
$ git clone git@github.com:mux-mux/coffeeHouse.git

# Go into the repository
$ cd coffeeHouse

# Install dependencies
$ npm install

# Start React App
$ npm start
```
